# API Endpoints

```text
GET  /api/health
GET  /api/file-config

POST /api/customers
GET  /api/customers
GET  /api/customers/{customer_id}

POST /api/enquiries
GET  /api/enquiries
PATCH /api/enquiries/{enquiry_id}/status

POST /api/files
GET  /api/files
GET  /api/files/{file_id}
GET  /uploads/{filename}

POST /api/invoices
GET  /api/invoices
GET  /api/invoices/{invoice_id}
PATCH /api/invoices/{invoice_id}

POST /api/payments
GET  /api/payments
GET  /api/payments/{payment_id}

POST /api/requirement-notes
POST /api/work-notes

GET  /api/dashboard
GET  /api/export/customers.csv
GET  /api/export/enquiries.csv
GET  /api/export/invoices.csv
GET  /api/export/payments.csv
```
